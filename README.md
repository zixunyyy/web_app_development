## 系統簡介

本專題開發一套簡易的任務管理系統，主要功能包含任務新增、編輯、刪除與狀態管理，讓使用者可以有效地規劃與追蹤日常工作或學習進度。

在現代生活中，許多人常因任務繁多而難以整理與管理，本系統透過清楚的介面與簡單的操作流程，協助使用者提升時間管理效率。

本系統的目標使用者為學生與一般使用者，特別適合用於作業管理、專題進度追蹤或日常待辦事項整理。說明你們做了什麼、解決了什麼問題、目標使用者是誰。

## 個人心得

楊紫薰

在這次任務管理系統的開發過程中，我學習到如何使用 Flask 建立基本的 Web 應用程式，並透過 SQLite 進行資料儲存與操作。同時，我也熟悉了 CRUD（新增、讀取、更新、刪除）功能的實作方式。

在開發過程中，我遇到最大的困難是在資料庫操作與前端畫面同步的部分，例如新增或刪除任務後，畫面沒有即時更新。後來透過重新導向（redirect）與模板渲染（render_template）的方式成功解決。

此外，我也學會如何將後端資料傳遞到前端頁面，並利用 Jinja2 動態顯示資料。這次專題讓我對 Web 開發流程有更完整的理解，對未來進一步學習全端開發有很大的幫助。
